功能亮点

单目标与批量目标检测（文件逐行读取）
将发现的目标写入输出文件，便于后续处理

环境与依赖

Python 3.x（推荐 3.8+）

第三方库（最少）：

pip install requests

（若使用增强版可考虑 urllib3、tqdm 等）

快速开始
查看帮助
python poc.py -h

单目标检测
python poc.py -u http://example.com

批量检测

python poc.py -f targets.txt

将目标按行放在 targets.txt（每行一个 URL），示例：
http://site1.com
http://site2.com
http://site3.com
